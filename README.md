An end-to-end data analytics project analyzing 3,900 customer shopping records using Python, PostgreSQL, and Power BI to uncover customer behavior, purchasing trends, and business insights.

# 🛍️ Customer Shopping Behavior Analysis

## 📌 Overview

This project is an end-to-end **Data Analytics** project that analyzes customer shopping behavior using a retail transaction dataset. The objective is to understand customer purchasing patterns, identify business opportunities, and transform raw data into meaningful insights.

The project follows a complete analytics workflow, starting with data loading and preprocessing in **Python**, performing **Exploratory Data Analysis (EDA)**, cleaning and preparing the data, analyzing it using **MySQL Server**, creating an interactive **Power BI dashboard**, documenting the findings in a report, and presenting the results through a professional presentation created with **Gamma**.

---

## 📂 Dataset

The dataset contains **3,900 customer purchase records** with information such as:

- Customer Demographics (Age, Gender, Location)
- Subscription Status
- Product Category
- Purchase Amount
- Discount Information
- Shipping Type
- Product Ratings
- Purchase Frequency

The data was cleaned and transformed before analysis to ensure accuracy and consistency.

## 📈 Project Workflow

### 1. Data Loading
- Imported the dataset into Python using Pandas.
- Explored the dataset structure and data types.

### 2. Exploratory Data Analysis (EDA)
- Examined missing values and duplicate records.
- Generated summary statistics.
- Explored customer demographics and purchasing behavior.
- Visualized important trends and distributions.

### 3. Data Cleaning
- Handled missing values.
- Standardized column names.
- Removed redundant data.
- Created additional features for analysis.

### 4. SQL Analysis (MySQL Server)
Performed SQL queries to answer business questions, including:
- Revenue by gender
- Customer segmentation
- Top-performing products
- Subscription analysis
- Shipping performance
- Age-group revenue analysis
- Repeat customer behavior

### 5. Power BI Dashboard
Developed an interactive dashboard to visualize key business metrics and customer insights.

### 6. Report & Presentation
- Documented the complete analysis in a project report.
- Created a professional presentation using **Gamma** to communicate key findings and recommendations.

## 🚀 How to Run

1. Clone this repository.
2. Open the Jupyter Notebook and install the required Python libraries.
3. Run the data preprocessing and EDA notebook.
4. Import the cleaned dataset into MySQL Server.
5. Execute the SQL queries.
6. Open the Power BI (.pbix) file to explore the interactive dashboard.
7. Refer to the project report and Gamma presentation for detailed insights.
